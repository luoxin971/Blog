云服务器初始化

云服务器

最近又买了台低配的轻量应用服务器，开始捣鼓，现在开始记录一下初期的全过程

添加SSH 登录

添加ssh登录主要分为三步：本地机器生成 ssh key，将公钥上传至服务器，修改服务器配置

本地生成 ssh key

如果之前有生成过 ssh key 且想继续使用的话，这一步可以跳过

生成 ssh key 命令如下

ssh-keygen -t ed25519 -C "your_email@example.com" -f ~/.ssh/ed25519_id

（可选）编写本地机器文件 ~/.ssh/config，添加以下内容

Host <connection_name>
    HostName <your_server_ip>
    User root
    IdentityFile ~/.ssh/ed25519_id

公钥上传至服务器

这一步介绍两个方法：

  • 在本地机器上
ssh-copy-id <connection_name>

<connection_name> 在 config 文件中设置，也可直接使用

ssh-copy-id <username>@<host>
  • 进入云服务器
vim .ssh/authorized_keys
sudo chmod 600 .ssh/authorized_keys

然后将上一步生成的公钥内容粘贴到 authorized_keys 里

修改服务器配置

vim /etc/ssh/sshd_config

修改配置

PasswordAuthentication yes # 改为no
PubkeyAuthentication yes # 将注释打开并改为yes
AuthorizedKeysFile	.ssh/authorized_keys  # 私钥路径
# 其它选项自己看文件注释按需修改

最后

systemctl restart sshd
# 也可以用 service sshd restart

潜在的坑：

  1. 如果是为 user 添加 ssh 登录，记得修改服务器里相关文件的 owner
  2. 如果修改了 ssh 的默认端口，记得要在服务器控制台里对外开放相应端口

创建用户

useradd -m <username>

如果要设置密码

passwd <username>

如果要删除用户

userdel -r <username>

添加 sudo 用户组

usermod -a -G sudo <username>

随后按照上述[添加SSH 登录](##添加SSH 登录)的内容为新用户设置 ssh 登录，注意相关文件权限

shell 相关

  • 安装基础命令

sudo systemctl restart networking

安装clash

试过了很多办法都不太行，最后还是简单粗暴的从本地上传文件到服务器，用到scp命令（貌似Windows自带）

scp -P <port> <local_file_path> <username>@<host>:<server_file_path>

首先在本地机器上下载 clash，然后用 scp 传输至服务器，然后解压

gzip -d <gz_file>
mv <filename> clash
chmod +x clash
# 下载订阅
wget -O config.yaml <subscribe_url>
./clash -d .

如果提示下载不来 Country.mmdb，则继续将本地机器上的文件用 scp 传输过去

注意，config.yaml 里设置 external-controller: '0.0.0.0:9090'，即可在 Clash-dashboard 进行远程设置

zsh, omz 安装

可根据 Installing ZSH来安装 zsh

sudo apt install zsh
zsh --version
chsh -s $(which zsh)
# 重新登录

根据 Installing OMZ来安装 omz

sh -c "$(wget -O- https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

然后根据 ~/.zshrc 文件按需修改配置即可

下面介绍两款插件 zsh-autosuggestions和zsh-syntax-highlighting

git clone https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions
# 接着在 .zshrc 文件的 plugins 里添加 zsh-autosuggestions
# 常用插件 plugins=(git z sudo zsh-autosuggestions)

git clone https://github.com/zsh-users/zsh-syntax-highlighting.git ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting
echo "source ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting/zsh-syntax-highlighting.zsh" >> ${ZDOTDIR:-$HOME}/.zshrc
source $HOME/.zshrc
