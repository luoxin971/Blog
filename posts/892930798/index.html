Windows 常用软件推荐

最近重新给电脑装了一个专业版的 Win10，很多软件又得重新下载，所以想着写篇文章记录一下，也方便自己之后类似情况直接用起来。

Daily

  1. 微信

  2. QQ

  3. Chrome

Tool

  1. 7z : 压缩文件
  2. PDF 阅读器（推荐 Adobe）
  3. AutoHotKey : 自定义快捷键
  4. DnsJumper : 开源的 DNS 服务器测试工具
  5. DropIt : 文件分类
  6. Everything : 文件索引
  7. TrayS : 任务栏美化工具，来自 52pojie
  8. PicGo : 文件上传，链接图床
  9. QuickLook : 文件预览
  10. Typora : MarkDown 文件编辑器
  11. V2rayN : 访问 Google 用
  12. Office

Game

  1. Steam

Programming

  1. Eclipse
  2. Git
  3. Maven
  4. JDK
  5. VS Code
  6. Navicat Premium 15
  7. NodeJs
  8. XSHELL
  9. Beyondsoft Compare 4 : 文件对比，激活
  10. Windows Terminal

还有 MySQL，XFTP 以及其他的开发环境就不说了。

以上基本就是本机电脑全部的软件了。

谢谢！
